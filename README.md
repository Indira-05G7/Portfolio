Welcome to my personal portfolio repository! This responsive, dark-theme-enabled website is built to showcase my skills, projects, educational background, and achievements as a B.Tech Computer Science student and aspiring Software Engineer.

## 🛠️ Built With
- **HTML5** for semantic structure
- **CSS3** for modern styling, animations, glassmorphism UI, and responsive layout
- **JavaScript** for interactivity (theme toggling, navigation, form submission)
- [EmailJS](https://www.emailjs.com/) for handling the contact form submissions directly from the client side
- [Font Awesome](https://fontawesome.com/) for scalable icons
- [Google Fonts](https://fonts.google.com/) (Plus Jakarta Sans & Outfit) for typography

## 📑 Sections Included
- **Home**: Brief introduction, objective, and quick navigation links.
- **About Me**: Details about my background, current studies, and technical focus areas.
- **Skills**: Categorized list of technical skills including Programming Languages, Web Technologies, Databases, and Tools.
- **Projects**: Highlights of my recent technical work:
  - Travel Itinerary Planner
  - Online Bookstore Application
  - Blood Bank Management System
  - Feast Forward
- **Education**: Timeline of my academic journey starting from Sri Vasavi Engineering College.
- **Certifications & Achievements**: Professional certifications (Oracle, Salesforce, NPTEL) and hackathon/competition participations.
- **Contact**: A fully functional contact form to get in touch directly.

## 📥 Local Setup
To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Indira-05G7/your-repo-name.git
   ```
2. Navigate to the project directory:
   ```bash
   cd your-repo-name
   ```
3. Open `index.html` in your web browser, or use a local development server like **Live Server** in VS Code to see changes in real-time.

> **Note on the Contact Form:**
> The contact form uses EmailJS. The EmailJS integration is already configured in the `index.html` file via `<script>` tags using a public key, service ID, and template ID.
